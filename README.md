欢迎来到 **北京航空航天大学自动寻航实践队** 的官方科创课程开源仓库！

本仓库致力于将高校优质的科创教育资源下沉，打造**低门槛、模块化、高可复制**的“教学交付套件”。无论你是偏远乡村的中学教师、支教团队，还是对前沿科技感兴趣的自学者，都可以通过本仓库获取完整的课程体系，将科创的火种播撒到更远的地方。

## 🌟 关于我们与“三端协同”模式

我们是一支由热爱科技与探索的北航青年组成的创新团队。结合国家“新质生产力”发展战略，我们探索出了一套 **“认知—供给—需求”** 三端协同的实践路径：
* **认知端（思想引领）**：挖掘红色文化，重构转型话语，凝聚奋斗与创新的社会共识。
* **供给端（科创下沉）**：依托“寻航·元知”平台与本开源仓库，提供包含软硬件的模块化课程，培养可迁移的技能与人才。
* **需求端（产业赋能）**：深入企业与地方，识别痛点，推动技术场景化落地。

本仓库正是我们 **“供给端”** 核心成果的开源集散地。

## 📚 课程模块与开源内容

我们把在山西友兰中学、中阳一中、太原五中，浙江瑞安中学、玉海实验中学等经过线下授课检验的精品课程进行了标准化打包。每个课程包内均包含：**课程教案**、**源码**、**物料清单** 以及 **授课指南**。

### 🛠️ 01. 智能硬件与工程实践 
注重“从0到1”的动手能力，将枯燥的电路与代码转化为看得见摸得着的智能小车。
* **[惯性导航地磁探矿小车](./01_Engineering_Hardware/Geomagnetic_Car/)**：模拟真实工业探矿痛点。包含底盘拼接、传感器原理、寻迹逻辑与代码烧录。
* **[L298N 电机驱动与控制](./01_Engineering_Hardware/Motor_Control/)**：以车间自动巡检为场景，深入剖析 L298N 模块原理，理解智能设备的“运动神经”。
* **[月球车虚实趣玩课](./01_Engineering_Hardware/Lunar_Rover/)**：结合 3D 打印与数实混合教学，体验航天工程的系统设计。

### 🧠 02. 人工智能与前沿视野 
打破信息差，拉升科技视野，探讨人机共存的哲学与未来。
* **[人类 VS 人工智能](./02_AI_and_Algorithm/Human_vs_AI/)**：深入浅出讲解卷积神经网络 (CNN)，通过模式识别应用探讨 AI 的优势与情感伦理局限。
* **[陆空协同机器人技术前沿](./02_AI_and_Algorithm/Air_Ground_Collaboration/)**：以“天眼”（无人机）与“地手”（无人车）的配合为例，普及系统工程中 `1+1>2` 的宏大逻辑。

### 🔐 03. 密码学与数学逻辑 (Math & Logic)
发现数学的“防御之美”。
* **[秘密的“分身术”：密码学入门](./03_Math_and_Logic/Cryptography/)**：从古代“虎符”、罗马密码到现代的“门限秘密共享”与区块链底层逻辑。

### 📡 04. 通信与物联网技术 
* **[WiFi 与蓝牙的“隔空传话”](./04_Communication_Tech/Wireless_Comm/)**：拆解通信协议，用通俗动画演示“沙砾变芯片”及数据传输的奇迹。

### 🎓 05. 生涯规划与成长 
* **[大学杂谈：思考未来](./05_Career_and_Vision/University_Talk/)**：拆解庞大工科专业图谱，助力中学生打破地域信息壁垒，完成从“做题”到“做人”的思维转变。

## 🛠️ 如何使用这些资源？ (以硬件课程为例)

为了方便您最快地把这门科创课开起来，我们将所有的资料都分门别类装在了不同的“抽屉”里。如果您对电脑编程不太熟悉，请不要担心，对照下面的 **“翻译表”** ，您就能轻松找到需要的材料：

| 电脑上的英文文件夹/文件名 | 翻译及内容 | 给老师的建议使用方法 |
| :--- | :--- | :--- |
| **`Lectures`** | **课件或讲义** | **直接点开。** 这是PPT或讲义。为了方便您在任何电脑上都能直接打开不乱码，我们提供了 PDF 版本。 |
| **`Instructor_Guide.md`** | **教案** | **备课必看！** 这是我们为您准备的上课剧本，里面写了怎么开场引出话题、哪里容易出错、怎么带学生互动。 |
| **`BOM.md`** | **材料采购清单** | **动手课必看** “BOM”就是物料清单。里面列出了上这门课需要买什么零件、买几个、大概多少钱。您照着清单网购即可。 |
| **`Code`** | **程序代码** | 这是让智能设备动起来的“指令”。我们在备课手册里会教您怎么把这些代码“复制粘贴”到小车主板里，您完全不需要从头学编程。 |
| **`Assets`** | **资源百宝箱** | 这里面装的是零碎的辅助材料，比如教学生怎么接电线的“彩色连线图”等。上课实操时打开让学生照着连线即可。 |
| **`README.md`** | **课程简介说明书** | 也就是您现在正在看的这份文档，为您介绍这门课的核心内容和准备工作。 |
| **`.gitkeep` 或 `.gitignore`** | **系统配置文件** | 这是给电脑系统看的隐藏文件，**您可以直接忽略它们**，完全不影响您的教学。 |。
